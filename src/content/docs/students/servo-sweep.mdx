## 🔌 接线图

| SG90 舵机 | Arduino |
|---|---|
| 棕色（GND） | GND |
| 红色（VCC） | 5V |
| 橙色（信号） | D9 |

> ⚠️ 如果舵机抖动或不动，说明 5V 供电不足——给舵机单独供电（5V 外接电源），GND 需共地。

## 💻 代码

```cpp
#include <Servo.h>

Servo myServo;
int angle = 0;

void setup() {
  myServo.attach(9);
}

void loop() {
  // 0° → 180° 慢扫
  for (angle = 0; angle <= 180; angle++) {
    myServo.write(angle);
    delay(15);
  }
  // 180° → 0° 慢扫
  for (angle = 180; angle >= 0; angle--) {
    myServo.write(angle);
    delay(15);
  }
}
```

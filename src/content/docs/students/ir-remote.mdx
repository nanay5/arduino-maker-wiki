## 💻 代码

> 先安装 IRremote 库：Arduino IDE → 工具 → 管理库 → 搜索 "IRremote" → 安装。

```cpp
#include <IRremote.h>

#define IR_PIN 2
#define RED 11
#define GREEN 12
#define BLUE 13

void setup() {
  pinMode(RED, OUTPUT); pinMode(GREEN, OUTPUT); pinMode(BLUE, OUTPUT);
  Serial.begin(9600);
  IrReceiver.begin(IR_PIN, ENABLE_LED_FEEDBACK);
  Serial.println("红外遥控器已就绪，按遥控器按钮");
}

void loop() {
  if (IrReceiver.decode()) {
    uint32_t code = IrReceiver.decodedIRData.decodedRawData;
    Serial.print("收到红外码: 0x");
    Serial.println(code, HEX);

    // 根据你的遥控器按键码修改以下值
    switch (code) {
      case 0xBA45FF00: digitalWrite(RED, !digitalRead(RED)); break;     // 按键1
      case 0xB946FF00: digitalWrite(GREEN, !digitalRead(GREEN)); break; // 按键2
      case 0xB847FF00: digitalWrite(BLUE, !digitalRead(BLUE)); break;   // 按键3
    }
    IrReceiver.resume();
  }
}
```

## ⚡ 上传与测试

1. 接好红外接收模块和三个 LED
2. 上传代码，打开串口监视器
3. 对准接收头按遥控器按钮，串口打印红外码（HEX 格式）
4. 把打印出的码替换到代码 switch 中

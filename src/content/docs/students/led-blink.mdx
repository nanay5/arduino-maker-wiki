## 💻 代码

```cpp
// 闪烁LED — 你的第一行 Arduino 代码
void setup() {
  pinMode(13, OUTPUT);  // 设置 D13 为输出
}

void loop() {
  digitalWrite(13, HIGH);  // 点亮 LED
  delay(1000);             // 等待 1 秒
  digitalWrite(13, LOW);   // 熄灭 LED
  delay(1000);             // 等待 1 秒
}
```

## ⚡ 上传与测试

1. USB 线连接 Arduino 到电脑
2. Arduino IDE 中：**工具 → 开发板 → Arduino Uno**
3. **工具 → 端口 →** 选择对应的 COM 口
4. 点击 **上传** 按钮（→ 箭头）
5. 观察板载 LED（D13）是否以 1 秒间隔闪烁

## 📊 直观对比

| | 普通 Scratch/Python 课 | 本 AI 创客课程 |
|---|---|---|
| **学什么** | 拖拽积木块 / 写代码 | 写代码 **+** 接电路 **+** 做机器人 |
| **输出** | 屏幕上的动画或游戏 | 能跑、能看、能听的实物小车 |
| **知识面** | 纯软件 | 编程 + 电子 + 机械 + AI |
| **家长能看到的成果** | 一个游戏链接 | 可以拍照发朋友圈的小车 |
| **学习门槛** | 会操作电脑即可 | 同样会操作电脑即可（我们会教接线） |
| **长期价值** | 培养编程思维 | 培养编程思维 **+** 动手能力 **+** 工程思维 |

## 🎯 为什么选择硬件+AI 方向？

- **看得见摸得着**：孩子做的不是虚拟动画，是真实的机器人
- **跨学科**：编程 + 物理（电路）+ 数学（距离/速度）+ 工程（结构）
- **AI 素养**：语音控制、视觉识别 → 理解 AI 如何与物理世界交互
- **竞赛通道**：可参加青少年科技创新大赛、机器人竞赛

## ⏱️ 90分钟标准课堂结构

| 环节 | 时长 | 核心目标 |
|---|---|---|
| 导入 | 5min | 展示成品效果，激发学生兴趣 |
| 原理讲解 | 10min | 讲清楚传感器/电机工作原理 |
| 安全提醒 | 5min | 接线安全、防短路、套件核对 |
| 动手搭建 | 30min | 学生看图接线、上传代码 |
| 调试优化 | 15min | 排查故障、修改参数 |
| 挑战延伸 | 15min | 分层任务——基础/进阶/高手 |
| 总结展示 | 10min | 点评作品、拍照记录 |

## 🔑 核心管理原则

- **课前**：套件齐全、电池有电、USB 线可用
- **课中**：编号管理、备件预案
- **课后**：拍照记录、清点回收
